# Model Risk Lab

Open-source framework for Model Risk Management, AI Governance and validation of statistical, machine learning and generative AI models.

---

## Vision

Create the first open-source framework that unifies validation methodologies for:

✔ Classical Statistical Models

✔ Machine Learning

✔ Large Language Models (LLMs)

✔ Generative AI

✔ AI Agents

under internationally recognised Model Risk Management principles.

---

## Why?

Financial institutions increasingly use Artificial Intelligence in critical decisions.

However, validation methodologies remain fragmented.

Model Risk Lab aims to provide a practical, extensible and transparent framework covering the complete validation lifecycle.
